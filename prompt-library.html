Writing & Email

Write a professional cold email

You are a B2B sales copywriter. Write a cold outreach email to [PROSPECT TITLE] at [COMPANY]. Our product [PRODUCT] helps [VALUE PROP]. Keep it under 120 words. Use a conversational tone. End with a low-friction CTA (not 'book a demo'). No buzzwords.

Rewrite for a different tone

Rewrite the following text in a [casual/formal/persuasive/friendly] tone. Keep the core message identical but change the voice completely. Preserve all facts and numbers. Text: [PASTE TEXT]

Generate blog post outline

Create a detailed outline for a 1,500-word blog post about [TOPIC]. Include: a compelling H1, 5-7 H2 sections, 2-3 bullet points per section describing what to cover, a conclusion with CTA, and 3 SEO-friendly title alternatives. Target audience: [AUDIENCE].

Summarize a long document

Summarize the following document in 3 levels: 1. One-sentence TLDR 2. Three-paragraph executive summary 3. Bullet-point key takeaways (max 10) Preserve all numbers, dates, and names exactly. [PASTE DOCUMENT]

Coding

Debug this code

I'm getting this error: [ERROR MESSAGE] Here's my code: ``` [PASTE CODE] ``` Explain what's wrong in plain English, then provide the fixed code with comments showing what changed and why.

Refactor for readability

Refactor the following code to improve readability, maintainability, and performance. Add docstrings/comments. Use modern best practices for [LANGUAGE]. Explain each change. ``` [PASTE CODE] ```

Generate API documentation

Generate API documentation for the following endpoint. Include: description, HTTP method, URL, request parameters (with types and required/optional), example request, example response, error codes, and a curl example. [PASTE CODE OR DESCRIBE ENDPOINT]

Marketing & SEO

Generate meta titles and descriptions

Generate 5 SEO-optimized meta title and description pairs for a page about [TOPIC]. Requirements: titles under 60 characters, descriptions under 155 characters, include primary keyword [KEYWORD], use action-oriented language, each pair should take a different angle.

Create a social media content calendar

Create a 2-week social media content calendar for [BRAND/PRODUCT] on [PLATFORM]. Include: post date, content type (carousel/video/text/story), caption text, hashtags (max 5), and best posting time. Tone: [TONE]. Goal: [GOAL]. Mix educational, entertaining, and promotional content 60/30/10.

Analyze competitor positioning

Analyze the positioning of these competitors: [LIST COMPETITORS]. For each, identify: their primary value proposition (one sentence), target audience, pricing strategy, key differentiators, and weaknesses. Then recommend how [MY PRODUCT] should position against them.

Data & Analysis

Analyze a CSV dataset

I'm uploading a CSV with [DESCRIBE DATA]. Please: 1. Summarize the dataset (rows, columns, data types, missing values) 2. Identify the top 3 most interesting patterns or trends 3. Flag any data quality issues 4. Suggest 3 visualizations that would tell the most compelling story 5. Write a 2-paragraph executive summary of the findings

Create a SWOT analysis

Create a comprehensive SWOT analysis for [COMPANY/PRODUCT] in the [INDUSTRY] market. For each quadrant, provide 4-5 specific, actionable points (not generic platitudes). Include a 'So What?' section that connects the SWOT findings to 3 strategic recommendations.

Productivity

Meeting notes to action items

Convert these meeting notes into a structured format: 1. Key decisions made (numbered) 2. Action items with owner and deadline (table format) 3. Open questions that need follow-up 4. One-paragraph summary suitable for sending to stakeholders who weren't present Meeting notes: [PASTE NOTES]

Create a project brief

Write a project brief for [PROJECT NAME]. Include: objective (one sentence), background/context (2-3 sentences), scope (in/out of scope), key milestones with dates, success metrics, risks and mitigations, team/roles, and budget estimate. Keep it to one page.

Photorealistic product shot

A [PRODUCT] photographed in a [SETTING] environment. Professional product photography, soft directional lighting, shallow depth of field, clean background with subtle [COLOR] tones. Shot on Canon EOS R5, 85mm f/1.4. 4K, ultra-detailed.

Consistent brand illustrations

A flat illustration in [BRAND COLOR 1] and [BRAND COLOR 2] showing [SUBJECT]. Minimal geometric style, no gradients, clean vector aesthetic, white background. Consistent with modern SaaS brand illustration style. Include subtle [ICON/ELEMENT] details.
